🔒 隱私政策

最後更新日期：2026 年 2 月 24 日

歡迎使用 Chingu（以下簡稱「本應用程式」）。我們非常重視您的隱私權。本隱私政策旨在說明我們如何收集、使用、儲存及保護您的個人資料。

1. 我們收集的資料

為提供服務，我們可能收集以下資料：

  • 帳號資料：電子郵件、姓名、出生日期、性別
  • 個人檔案資料：大頭照、自我介紹、興趣標籤、所在城市/行政區
  • 使用資料：配對偏好、活動參與紀錄、聊天訊息
  • 裝置資料：裝置型號、作業系統版本、推播通知 Token

2. 資料使用方式

我們使用收集的資料於以下用途：

  • 提供並改善配對與社交功能
  • 安排及管理晚餐活動
  • 發送推播通知（新配對、新訊息、活動提醒）
  • 偵測與防範濫用行為（檢舉、封鎖）
  • 分析應用程式效能與穩定性（Crashlytics）

3. 第三方服務

本應用程式使用以下第三方服務處理資料：

  • Firebase Authentication：帳號驗證與身分管理
  • Cloud Firestore：資料儲存
  • Firebase Storage：圖片儲存
  • Firebase Cloud Messaging：推播通知
  • Firebase Crashlytics：當機報告與效能監控
  • Google Sign-In：第三方登入（選用）

這些服務依照 Google 隱私政策 處理您的資料。

4. 資料儲存與安全

  • 您的資料儲存在 Google Firebase 的安全伺服器上
  • 所有資料傳輸皆使用 HTTPS 加密
  • 我們實施 Firestore 安全規則限制資料存取
  • 密碼經由 Firebase Authentication 加密處理，我們無法讀取您的密碼

5. 您的權利

您對個人資料享有以下權利：

  • 查看：您可隨時在「設定 → 編輯個人資料」中查看您的資料
  • 修改：您可隨時更新您的個人檔案資訊
  • 刪除：您可在「設定 → 隱私設定 → 刪除帳號」中永久刪除您的帳號與所有相關資料
  • 撤回同意：您可隨時停用推播通知或刪除帳號

6. 兒童保護

本應用程式僅限 18 歲以上使用者使用。我們不會故意收集 18 歲以下未成年人的個人資料。若我們發現有未成年人的資料被收集，將立即予以刪除。

7. 政策變更

我們可能不定期更新本隱私政策。重大變更時，我們會透過應用程式內通知或電子郵件告知您。繼續使用本應用程式即表示您同意更新後的政策。

8. 聯絡我們

若您對本隱私政策有任何疑問，請透過以下方式聯繫我們：
