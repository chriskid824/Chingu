# 產品核心:盲盒式 6 人晚餐

**Chingu 是什麼、不是什麼。所有功能決策先回到這頁。**

## 一句話定位

對標 Timeleft:「每週一場官方晚餐,5~7 個陌生人一桌,見面前零資訊」。不是交友軟體,是「信任演算法、擁抱未知」的真實社交工具。與 Timeleft 錯峰:Timeleft 週三、**Chingu 週四晚餐**。MVP 僅限台北市信義區。

## 產品三大鐵律(不可違反)

1. **拒絕事前篩選** — 無 Explore、無 Matching、無配對卡片。飯友由系統 100% 分配。相關模組已在 Phase 1 徹底刪除(非隱藏)。
2. **聊天靠實體解鎖** — 一對一私訊需晚餐後雙盲互評、雙向 👍 才開通;群組聊天於完全解鎖時(週三 17:00)自動建立。禁止任何「付費解鎖聊天」。
3. **時間軸驅動 UI** — 首頁是隨時間自動變化的動態儀表板(見 [時間軸狀態機](時間軸狀態機.md)),不是靜態頁面。

## 使用者核心旅程

```
報名本週晚餐 → 等待配對(電子邀請卡)
→ 週二 18:00 解鎖飯友輪廓(星座/產業/年齡段,幾何頭像)
→ 週三 17:00 解鎖餐廳地址 + 群組聊天開通
→ 週四晚間:5~7 人實體見面 + 破冰話題包
→ 週四 19:00:群組聊天內照片全部解鎖
→ 週五起 72 小時:雙盲 👍/👎 互評
→ 雙向 👍:解鎖一對一私訊聊天室
```

## 部分解鎖的資訊紅線

週二「部分解鎖」**只能**顯示:星座、產業 Icon、年齡段(如 25-30)。**絕對禁止**:真實姓名、照片/頭像、具體年齡、社群帳號連結。`gender` 與 `diningPreference` 在前端任何地方都不顯示給其他用戶。

## MVP 不含(勿誤做)

付費機制/Paywall(MVP 免費)、個性測驗(用興趣+基本資料簡易配對)、+1 邀請機制、成就徽章。
注意:`lib/models/subscription_model.dart`、`lib/screens/subscription/` 已存在骨架,但 MVP 不啟用付費。
