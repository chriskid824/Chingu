## 全域慣例

- 集合名稱 snake_case;欄位以 `lib/models/*.dart` 的 fromJson/toJson 為準,勿自創
- 色彩一律 `AppColorsMinimal`;文案繁體中文;commit message 中文
- `timeleft_pivot_plan.md` 是歷史決策紀錄,部分設計(Apple 橘色系)已被 CLAUDE.md 取代——引用前先確認版本先後
- 各頁行號是撰寫當下參考,**grep 關鍵字才是穩定錨點**;每頁末「AI 檢索與驗證」是給 AI 用的查證步驟
- origin 的 300+ 殭屍分支不可撿程式碼(部分違反產品鐵律),事實以 main 為準
