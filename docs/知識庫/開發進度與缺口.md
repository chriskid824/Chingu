- **rules 硬化**:dinner_events update 限自己報名/取消、dinner_groups 限參與者改出席、dinner_reviews create 驗 reviewerId、chat_rooms create 收回 CF
- **UI/動效**:FlipCard 餐廳翻牌、BounceButton/AnimatedCounter/Skeleton/stagger 接線、27 處硬編碼色收斂、morandiPalette token
- 測試 116 → 128 全綠(補 P0:首頁狀態機、雙盲不變量、聊天權限閘門)
