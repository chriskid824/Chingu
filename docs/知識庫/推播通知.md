## Cloud Functions 分工

| 檔案 | 職責 |
|---|---|
| `scheduledNotifications.ts` | 時間軸定時推播 + 72 小時逾期評價自動跳過 |
| `pushNotifications.ts` | trigger 型:新訊息、雙向 Match |
| `sendBroadcast.ts` | 管理員手動廣播(可按城市/用戶/全部定向) |
| `notification_content.ts` | 推播文案範本(改文案改這裡,勿散落各處) |

## App 端

- `lib/services/push_notification_service.dart` — FCM 初始化、token、前景處理
- `lib/services/rich_notification_service.dart` — 富文本通知
- `lib/services/badge_count_service.dart` — App icon 徽章
- `lib/models/notification_model.dart` + Firestore `notifications` 集合(create 僅 Cloud Functions,本人可讀/標已讀)
- 設定頁:`notification_settings_screen.dart`、權限引導:onboarding 的 `notification_permission_screen.dart`
- 套件:firebase_messaging 15.1.6 + flutter_local_notifications 18.0.1
