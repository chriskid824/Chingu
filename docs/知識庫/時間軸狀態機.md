# 時間軸狀態機:一週流程與首頁狀態

**全 app 的心臟:每週固定時間軸驅動 DinnerEvent/DinnerGroup 狀態與首頁 UI。**

## 每週時間軸(Cloud Functions 排程實況,2026-07-11 核對)

所有排程都設了 `timeZone("Asia/Taipei")`,下表時間即台北時間:

| 時間 | 函數 | 動作 | 狀態變化 |
|---|---|---|---|
| 週二 00:00 | createWeeklyEvent | 建立本週四 DinnerEvent(**規格寫週一,實作是週二**,報名窗僅 12hr) | event `open` |
| 週二 08:00 | sendSignupReminder | 推播全體活躍用戶「開放報名」 | — |
| 週二 12:00 | processWeeklyGrouping(weeklyScheduler.ts) | 截止+分桌+指派餐廳 | event `closed`,產出 dinner_groups(`pending`) |
| 週二 18:00 | revealCompanions | 寫入匿名同伴卡+推播 | group `info_revealed` |
| 週三 17:00 | revealRestaurants(scheduledNotifications.ts) | 建群組聊天+推播餐廳 | group `location_revealed`,event `revealed` |
| 週四 18:00 | sendDinnerReminder | 推播「今晚見」 | — |
| 週四 19:00 | unlockPhotos | 群聊照片解鎖(寫 participantAvatars + photosUnlocked) | — |
| 週四 22:00 | completeEvents | 記 completedAt | event/group `completed` |
| 週五 10:00 | sendReviewReminder | 評價提醒 | — |
| 週日 10:00 | sendReviewUrgentReminder | 「還有 24 小時」 | — |
| 週一 10:00 | autoSkipReviews | completedAt 滿 72hr 的未評價自動 `skipped` | group `reviewStatus='completed'` |

**時區地雷(2026-07-11 已修)**:排程的 cron 有設台北時區,但函數內部 `new Date()`/`setHours()` 跑在 UTC 容器,曾導致 eventDate/signupDeadline 全錯 8 小時(截止時間建立當下就已過期)。現在所有牆鐘計算集中在 `functions/src/taipeiTime.ts`,**任何新排程禁止直接用 setHours/getDay**。時間一律 UTC 儲存,App 端轉時區。

## 首頁狀態(動態儀表板)

| 狀態 | 觸發條件 | UI |
|---|---|---|
| 未報名 | 用戶未報名本週 | 倒數圓環「距截止 X 天」+ CTA「立刻報名」 |
| 配對中 | 已報名待配對 | 電子邀請卡 + 脈衝動畫 |
| 部分解鎖 | 週二 18:00 後 | 邀請卡翻轉,飯友模糊卡片(星座+產業+年齡段) |
| 完全解鎖 | 週三 17:00 後 | 餐廳地圖卡片 + 導航 + 破冰話題包入口 |

注:實作為「5 狀態」(commit 62a7c02「首頁 5 狀態動態儀表板」),比規格 4 狀態多一個;細節 grep `lib/screens/home/`。有首頁狀態機單元測試(commit 7a1f69d)。

## 照片顯示規則

- Onboarding 照片選填;未上傳者用系統分配的**抽象幾何頭像**(`GeometricAvatar`,commit 059e74e,grep `PhotoVisibility`)
- 見面前一律幾何頭像(部分解鎖也不顯示照片)
- 週四 19:00:群組聊天室內照片解鎖
- 一對一聊天室(Match 後):直接顯示真實照片

## 破冰話題包

- 來源:Firestore `icebreaker_questions` 動態載入,每場隨機 10 題
- 3 層級:暖場(前 3)→ 深入(中 4)→ 靈魂拷問(後 3)
- UI:卡片式一次一題、左右滑動、大字(餐桌上使用);週三完全解鎖後出現入口;隔天可在往期回顧回看

## 取消與不足額規則(邊緣案例)

- 週二中午後**禁止取消**;之前可自由取消
- 成團 5~7 人彈性;不足 5 人本週取消、保留至下週(發「下期免費券」+ 溫暖文案)
- 72 小時未評價自動視為「跳過」
- 週三解鎖高併發:App 端本地快取 + 分批推播(間隔 3 秒)
