# App Store Optimization (ASO)

## 概述
完整的 App Store 優化工具包，用於研究、優化和追蹤 Apple App Store 和 Google Play Store 上的行動應用程式效能。

## 能力

### 研究與分析
- 關鍵字研究和競爭分析
- 市場趨勢識別
- 競爭對手追蹤
- 搜索量估算

### 元資料優化
- 應用名稱和標題優化
- 副標題/短描述優化
- 完整描述寫作
- 關鍵字欄位優化（iOS）
- 截圖和視頻策略

### 轉換優化
- A/B 測試規劃
- 圖示設計指導
- 截圖順序優化
- 預覽視頻建議

### 評分與評論管理
- 評論情緒分析
- 回應策略
- 評分改善策略

## Chingu App 專屬 ASO 策略

### 目標關鍵字

**主要關鍵字（台灣市場）:**
- 交友 App
- 約會 App
- 聊天交友
- 認識新朋友
- 單身交友
- 飯友
- 社交 App

**長尾關鍵字:**
- 興趣配對交友
- 找飯友 App
- 台北交友
- 有品質的交友 App

### App Store 列表優化

#### 應用名稱
```
Chingu - 興趣配對交友
```

#### 副標題
```
找到志同道合的飯友
```

#### 關鍵字欄位（iOS - 100字符限制）
```
交友,約會,聊天,飯友,社交,配對,單身,認識朋友,興趣,美食,餐廳
```

### 截圖策略

建議 5-6 張截圖，順序如下：

1. **英雄截圖** - 展示核心價值主張
   - 文字：「找到志同道合的朋友」
   
2. **配對功能** - 展示智慧配對
   - 文字：「AI 智慧配對」
   
3. **聊天功能** - 展示對話介面
   - 文字：「輕鬆開始對話」
   
4. **飯局功能** - 展示約飯功能
   - 文字：「一起享受美食」
   
5. **安全功能** - 展示驗證機制
   - 文字：「真實用戶驗證」

### ASO 評分清單

| 項目 | 權重 | 檢查內容 |
|-----|------|---------|
| 關鍵字相關性 | 25% | 標題和描述是否包含目標關鍵字 |
| 視覺吸引力 | 20% | 圖示和截圖品質 |
| 描述品質 | 15% | 描述是否清晰傳達價值 |
| 評分和評論 | 20% | 應用評分 ≥ 4.0 |
| 更新頻率 | 10% | 定期更新 |
| 本地化 | 10% | 針對目標市場本地化 |

### 最佳實踐

#### 關鍵字研究
- 每月審查關鍵字表現
- 監控競爭對手關鍵字變化
- 根據季節和趨勢調整

#### 元資料優化
- 在標題和副標題中使用主要關鍵字
- 描述前幾行最重要（折疊前可見）
- 使用項目符號列出功能

#### 視覺資產
- 圖示應簡潔且在小尺寸下可辨識
- 截圖使用設備框架和文字覆蓋
- 考慮製作預覽視頻

#### 評論管理
- 24-48 小時內回應所有評論
- 對負面評論提供解決方案
- 引導滿意用戶給予評分
